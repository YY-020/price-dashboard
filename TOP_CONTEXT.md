# 市场智能体 · 业务背景

## 一、行业背景

**所属行业**：锂离子电池电解液及上游原材料制造业。

**核心业务范围**：
- 电解液、锂盐、溶剂、添加剂、磷酸铁锂、碳纳米管（CNT）等关键材料的市场研究与战略分析。
- 重点聚焦欧美市场动态（政策、产能、客户、竞对），兼顾中国及全球产业链。

**核心信息来源**：
- 付费数据库：百川盈孚（BaiInfo）、上海有色网（SMM），由分析师每周手动下载 Excel 更新。
- 内部采购数据：GEO 内部采购报价，由采购部定期提供。
- 海外行业媒体：Electrive、Battery-News-DE、Thelec、MarkLines 等，通过 RSS 或网页爬虫自动采集。
- 微信公众号：高工锂电、起点锂电、我的电池网等行业公众号，目前为人工采集。

**核心信息处理流程**：
- 价格数据 → 数据清洗 → 价格看板可视化。
- 行业新闻 → 采集 → 筛选 → 整理 → 日报/周报生成。

## 二、目标用户

| 用户 | 需求 | 使用场景 |
|:---|:---|:---|
| **市场分析师（本人）** | 高效完成日报/周报，快速查询价格和新闻，辅助深度调研 | 每日开工、周五周报、临时调研需求 |
| **市场部同事** | 查看价格走势，下载数据，查询市场动态 | 需要数据支撑决策时，自行登录看板查询 |
| **领导层** | 快速了解市场变化趋势，支撑战略决策 | 查看日报/周报摘要，或直接登录看板 |
| **未来外部客户** | 获取行业数据和报告 | 购买商业化产品后登录使用 |

## 三、技术环境约束

**网络环境**：
- 中国大陆办公网络，Google 搜索不可用。
- 访问部分海外网站（如 Battery-News-DE）可能较慢，但总体可访问。
- 海外 AI 工具（如 Claude Code、ChatGPT、OpenAI API）需翻墙，不适合当前阶段大规模使用。

**成本控制原则**：
- 优先使用免费/国产工具：DeepSeek、元宝、Trae（国内版）、扣子、豆包。
- 避免海外付费 API，除非明确无替代方案且价值极高。

**数据安全原则**：
- 价格数据来源于付费数据库，需注意合规使用，不对外公开发布。
- 优先本地或内网部署，不暴露到公网。
- 若未来上云，需配置账号密码登录和权限管理。

**可维护性要求**：
- 整套系统未来可能交接给市场团队其他同事。
- 代码需有清晰注释，文档需完整，操作需尽量"傻瓜式"（双击启动）。

## 四、业务术语

| 术语 | 说明 |
|:---|:---|
| **电解液** | 锂离子电池四大核心材料之一，由锂盐、溶剂、添加剂配制而成 |
| **六氟磷酸锂（LiPF6）** | 目前最主流的电解液锂盐 |
| **LiFSI** | 新型锂盐，双氟磺酰亚胺锂，性能优于LiPF6，成本更高 |
| **VC（碳酸亚乙烯酯）** | 电解液添加剂，用于提高电池循环寿命 |
| **FEC（氟代碳酸乙烯酯）** | 电解液添加剂，用于改善低温性能 |
| **CNT（碳纳米管）** | 导电剂，用于提高电池正负极导电性 |
| **LFP（磷酸铁锂）** | 正极材料，用于磷酸铁锂电池 |
| **NCM/NCA** | 三元正极材料，镍钴锰/镍钴铝 |
| **GWh / TWh** | 电池产能单位，吉瓦时 / 太瓦时 |
| **EV / BESS / ESS** | 电动汽车 / 电池储能系统 / 储能系统 |
