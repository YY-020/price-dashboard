# 锂电价格看板 UI 设计方案

## 一、项目背景

用户是电解液/锂电池行业市场分析师，需要做一个**锂电产业链材料价格看板**。当前版本功能基本可用但 UI 简陋，需要参照以下 17 张专业看板截图，生成一个统一的高保真设计方案。

### 目标用户
- 行业分析师，每天查看多种材料价格走势和对比
- 需要快速感知价格变动、拐点、涨跌幅

### 整体风格定位
- **深色专业金融终端风**（参考截图 14、16）
- 干净、信息密度适中、不花哨
- 中英文可切换

## 四、右侧展示区 — Tab 2：多材料对比

### 4.1 核心问题：材料多时线条重叠
用户场景：2-5 个材料对比（碳酸锂 vs 六氟磷酸锂、5 个溶剂、六氟 vs 双氟等）

### 4.2 解决方案（多选一或组合）

**方案 A：归一化百分比对比（强烈推荐）**
- 参考：**截图 01（Google Market Summary）**
- 所有材料以起始日为基准 100%，显示相对涨跌幅百分比
- 纵轴：-10% ~ +20%，线条不重叠
- 右侧带图例卡片列表（参考截图 01 右侧的指数卡片：图标 + 名称 + 当前值 + 涨跌幅）

**方案 B：双 Y 轴**
- 参考：**截图 08（Tableau 聚碳酸酯，多国家多线）**
- 价格量级差异大的材料用左右双 Y 轴
- 适合 2-3 个材料

**方案 C：分面小图（Small Multiples）**
- 每个材料一个独立小图，共享同一 X 轴时间
- 参考截图 11（G-Store 的多图表布局）
- 适合 3-5 个材料，便于独立观察

**推荐默认用方案 A，用户可切换方案 B/C。**

### 4.3 材料选择器
- 参考截图 15（Price Trend Analysis 顶部筛选行）
- 多选下拉框，带颜色标记
- 已选材料显示为彩色 pill 标签，可点击移除
- 最多同时选 5 个（超限提示）

### 4.4 图例设计
- 参考截图 01 右侧：每个材料一行，带彩色圆点/图标 + 名称 + 当前值 + 涨跌幅
- 点击图例可隐藏/显示该材料线条
